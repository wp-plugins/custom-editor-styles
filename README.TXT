== Description ==

Allows the user to easily apply classes to text within the visual editor from predefined styles/classes entries.

== Installation ==

1. Upload the entire custom-editor-styles folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Go to "Editor Styles" menu option and add styles and the corresponding class names.

== Frequently Asked Questions ==

= From where can i select the appropriate style? =

From the "Styles" dropdown menu of the visual editor
